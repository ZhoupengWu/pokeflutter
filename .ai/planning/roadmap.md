## ✅ Phase 3 — Polish & Optimization (v1.3.0)

### Goal
Optimize performance, add persistence, and complete the quiz feature.

### Completed
- ✅ **Caching System** — `lib/utils/pokemon_cache.dart`
  - Singleton `PokemonCache` with in-memory `Map<String, Pokemon>`
  - `PokemonApi.getPokemonDetails` checks cache before hitting the network
  - Zero extra dependencies — drop-in ready to be extended with Hive/Isar

- ✅ **Persistent Favourites** — `lib/utils/favourites_manager.dart`
  - `shared_preferences` added to `pubspec.yaml`
  - `FavouritesManager.init(allPokemon)` called in `main()` before `runApp`
  - `toggle()` is now `async` — writes to disk after every change
  - `ValueNotifier` still drives UI updates reactively

- ✅ **Quiz Feature** — `lib/views/quiz_page.dart`
  - 10 questions generated by `QuizLogic.generateQuestions()`
  - Mode: silhouette (black sprite) → guess the name from 4 options
  - Animated option buttons: green ✓ / red ✗ on answer
  - Progress bar coloured by current Pokémon type
  - Results screen with accuracy %, score circle, Play Again button
